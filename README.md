# CommandFinder

A console utility for easily searching commands in history

### To download the utility you need

1. U need to download installer
```shell
curl -O https://raw.githubusercontent.com/Dan-Egorov/CommandFinder/refs/heads/main/Mac_Linux_installer.sh
```

2. U need to start instalation
```shell
bash Mac_Linux_installer.sh
```

3. U need to choose place for downloading


4. End, you are download utility!!!!
